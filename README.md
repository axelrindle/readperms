# readperms
> Node cli app for reading UNIX file permissions.

## Install
```bash
$ npm install -g readperms
```
or
```bash
$ yarn global add readperms
```

## Usage

```bash
$ readperms [options] <files/dirs...>
```

For help type:
```bash
$ readperms --help
```
